# GinkoFrontend

> Submodule to be run by the GingkoServer

Credits: Imogen & Sophie (Philip)

---

## Features

- Sequential text areas for visitor input
- The second question is disabled until the first is submitted
- After the second submission, the first question updates dynamically
- Dark-mode UI optimized for exhibition screens
- Built entirely with **shadcn/ui** components (Textarea, Button)
- Easily extensible — add more questions or components with shadcn

---

## Prerequisites

- Node.js >= 18
```
scoop install main/nodejs
```
- pnpm
```
npm install --global corepack@latest
```
```
corepack enable pnpm
```
```
corepack use pnpm@latest
```

---

## Getting Started

1. **Install libs**
```
pnpm install
```
2. **Run the app locally**
```
pnpm dev
```
3. **Build for Production**
```
pnpm build
```
